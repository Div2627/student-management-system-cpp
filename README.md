# Student Management System (C++)

## Description
A console-based application developed in C++ to manage student records.
The system allows users to add, view, search, and delete student information
using structured programming concepts.

## Features
- Add new student records
- Display all students
- Search students by ID
- Delete student records
- Menu-driven interface

## Technologies Used
- C++
- Structures
- Arrays
- Functions
- Standard Input/Output

## Learning Outcomes
- Improved understanding of C++ fundamentals
- Hands-on experience with structures and arrays
- Better problem-solving and logical thinking
